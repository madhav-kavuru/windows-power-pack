# Traditional Software Innovation

Time and cost estimates for developing all 5 utilities in the Power Pack.

## Separate-team estimates

| Utility (separate team) | Conservative calendar time to polished v1 | Approx. team size | Conservative team burn rate (per month) | Conservative total cost |
| --- | --- | --- | --- | --- |
| Safe Drop Area for Explorer | 6–9 months | 2 dev + shared QA/design | $55k–$80k | **$330k–$720k** |
| Desktop open-folder highlighter | 5–8 months | 2 dev + shared QA/design | $55k–$80k | **$275k–$640k** |
| Browser Tab Name Display (BTND) | 5–8 months | 2 dev + shared QA/design | $55k–$80k | **$275k–$640k** |
| Filename Printer | 7–10 months | 2–3 dev + shared QA/design | $70k–$110k | **$490k–$1.1M** |
| Automatic pointer locator after idle | 3–5 months | 1–2 dev + shared QA/design | $40k–$70k | **$120k–$350k** |

## Portfolio result with 5 teams

Because all five teams run in parallel:

- **Overall calendar time:** driven by the slowest project (Filename Printer) → about **7–10 months** from start to all five shipped at a polished, widespread-distribution quality bar.
- **Total conservative cost across all five:** add the ranges above → about **$1.5M to $3.45M**.

This is a maximally parallel, highest-spend option, rather than the lower-cost alternative of using shared teams. It also assumes US-style fully loaded costs. Even with contingencies and overruns, the total is unlikely to exceed $4 million.

As can be seen, all 5 utilities in the Power Pack can be developed to a polished, ready-for-widespread-distribution quality level in less than a year for less than $4 million. Compared to the billions of dollars Microsoft is pouring into AI with uncertain ROI, this is an insignificant amount of money that can create tangible, long-lasting user benefits and incremental revenues through increased subscriptions to Microsoft 365 that far exceed the cost.

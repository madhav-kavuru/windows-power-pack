# windows-power-pack
Windows Power Pack: Five Small Utilities, Big Everyday Wins

This document is a concise summary; links under each utility point to full technical proposals and mockups for reviewers who want implementation-level detail.

This pack proposes five focused utilities that smooth common Windows workflows without changing how users fundamentally work. Each idea is small in scope but aimed at high-friction, high-frequency pain points professionals hit every day.

## Utilities

### 1. Safe Drop Area for Windows File Explorer

A File Explorer feature that adds a pinned, protected drop target at the top of crowded folder views so users can drag files into the current folder without accidentally dropping onto a visible subfolder.

Full spec and mockups: [Safe Drop Area details](docs/safe-drop-area-for-windows-file-explorer.md)

### 2. On-demand desktop open-folder highlighter

A Windows utility that temporarily highlights desktop folders that are currently open in File Explorer, helping users quickly see which desktop folders are already active before reorganizing icons or opening duplicates.

Full spec and mockups: [Desktop Open-folder Highlighter details](docs/desktop-open-folder-highlighter.md)

### 3. Browser Tab Name Display (BTND)

A taskbar-adjacent Windows 11 utility that shows the title of the active browser tab in a readable strip and lets the user step left or right through tabs while seeing each tab name clearly.

Full spec and mockups: [Browser Tab Name Display details](docs/browser-tab-name-display-(BTND).md)

### 4. Filename Printer

A Windows print-pipeline utility that inserts a removable filename reference page at print time so physical printouts can be traced back to their source files without modifying the original document.

Full spec and mockups: [Filename Printer details](docs/filename-printer.md)

### 5. Automatic pointer location after idle

A small enhancement to Mouse Properties that automatically shows the existing pointer-location animation after a user-configurable idle period, making it easier to find the pointer when returning to the PC.

Full spec and mockups: [Automatic Pointer Location After Idle details](docs/automatic-pointer-locator-after-idle.md)
